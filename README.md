# consumer

## 项目介绍

> 项目名称：鲜橙优鲜\_C 端

> 简介:购物商城类 App 移动端,移动端适配采用 lib-flexible+postcss-px2rem 方法

> 技术栈：Vue 全家桶+axios+vant+less

> 开源地址：一个月前左右写的一个小项目，先开源于 github https://github.com/liuyuanzhi-xx/vue_mall_C.git

## 项目安装

```
npm install
```

### 项目运行

```
npm run serve
```
